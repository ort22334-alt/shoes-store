# Каталог обуви

Запускается на [Render.com](https://render.com) автоматически.

## Как задеплоить

1. Зарегистрируйся на [https://render.com](https://render.com)
2. Создай новый **Web Service** → **Connect GitHub Repository**
3. Выбери этот репозиторий
4. Render автоматически прочитает `render.yaml` и запустит сайт
5. Через 2–3 минуты получишь ссылку вида: `https://shoes-store.onrender.com`

✅ Готово
